# Pure API AI Assistant without MCP / with MCP

🎯 Features
Your CLI assistant will:
    - Fetch Weather (Weather API)
    - Read Local File
    - Query PostgreSQL
    - Use LLM to interpret user input
    - Manually route logic using if-else

🎯 Prerequisites
    - Python 3.12
    - Weather API credentials
    - PostgreSQL credentials
    - LLM credentials
